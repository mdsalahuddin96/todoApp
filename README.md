এ প্রজেক্টিতে একটি ইভেন্ট মার্ক করা এবং ডিলিট করার জন্য আমি Event Delegation পদ্ধতি ব্যবহার করেছি।
Event Delegation হলো Parent Node এ Event যুক্ত করে child node কন্ট্রোল করার একটি পদ্ধতি। 
এটি খুবই কার্যকরী একটি পদ্ধতি কারণ এর মাধ্যমে ডায়নামিক্যালি যুক্ত হওয়া এলিমেন্টগুলো সহজে হ্যান্ডেল করা যাচ্ছে।
তাছাড়া যেহেতু শুধু target event টি সিলেক্ট করে তাতে কাজ করছি তাই মেমরিও কম খরচ হচ্ছে ।
